# task-manager-app
Responsive task manager built with HTML, CSS, and JavaScript featuring dark mode, priority sorting, due dates, and Local Storage persistence.
## Features
- Add, edit, and delete tasks
- Mark tasks as completed
- Filter tasks (All, Completed, Pending)
- Task priority (High, Medium, Low)
- Due date support
- Dark mode toggle
- LocalStorage persistence
- Smooth UI animations

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the project
2. Open index.html in your browser
